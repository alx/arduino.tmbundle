The Arduino Textmate Bundle.

<h2>Installation</h2>

* Run this:
 
<pre>
cd ~/Library/Application\ Support/TextMate/Bundles
git clone git://github.com/alx/arduino.tmbundle.git Git.tmbundle
</pre>

* Only one shortcut is available to make and upload your program on the Arduino board: Command-Shift-A

<h2>Support</h2>

<a href="http://github.com/alx/arduino.tmbundle/">GitHub Repository</a>.

<h2>Troubleshooting</h2>

<p>"<b>error: stray '\1' in program</b>": delete the whitespace after the method name if there is one.</p>
